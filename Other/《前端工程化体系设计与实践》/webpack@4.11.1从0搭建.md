# webpack@4.11.1从0搭建

> 为什么webpack这么难

- 可能并不是十分清楚一个前端项目需要哪些构建服务
- webpack里插件和loader的写法经常有2、3种写法，有字符串、数组、对象、单独文件配置
- node.js和npm的版本、webpack版本、相关插件的版本变动

解决方法就是只有一个，看官方文档，并且每一行都自己去配置。

## 首先思考一个项目，一般在构建环节会做哪些事情？

- 语言层面编译
    + css的预编译
    + es6语法的转换
    + vue模板语法、jsx语法需要编译成JS
- 面向性能
    + 依赖打包
    + 文件合并
    + 资源嵌入
    + hash指纹
- 面向部署
    + 打包后，路径的调整
    + 文件名hash
